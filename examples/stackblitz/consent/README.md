# NDPR Consent — StackBlitz demo

Minimal Next.js 15 + React 19 scaffold that boots the `NDPRConsent` preset
from `@tantainnovative/ndpr-toolkit`. The page renders a single page with the
consent banner anchored at the bottom; saved categories are logged to the
browser console.

Covers NDPA 2023 Section 26 (informed, specific, freely-given consent).

- Preset: `NDPRConsent` from `@tantainnovative/ndpr-toolkit/presets/consent`
- Styles: imported once in `app/layout.tsx`
- Run locally: `npm install && npm run dev`

Open in StackBlitz:
https://stackblitz.com/github/mr-tanta/ndpr-toolkit/tree/main/examples/stackblitz/consent
