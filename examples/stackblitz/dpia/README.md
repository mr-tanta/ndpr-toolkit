# NDPR DPIA — StackBlitz demo

Minimal Next.js 15 + React 19 scaffold that boots the `NDPRDPIA` preset.
Walks through a multi-section Data Protection Impact Assessment questionnaire
seeded with sensible NDPA defaults; the final answers map is logged to the
browser console when the wizard completes.

Covers NDPA 2023 Section 28 (mandatory DPIA for high-risk processing).

- Preset: `NDPRDPIA` from `@tantainnovative/ndpr-toolkit/presets`
- Run locally: `npm install && npm run dev`

Open in StackBlitz:
https://stackblitz.com/github/mr-tanta/ndpr-toolkit/tree/main/examples/stackblitz/dpia
