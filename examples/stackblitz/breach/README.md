# NDPR Breach Report — StackBlitz demo

Minimal Next.js 15 + React 19 scaffold that boots the `NDPRBreachReport`
preset. Internal-facing form your DPO uses to capture the details of a
personal-data breach; the submitted report is logged to the browser console.

Covers NDPA 2023 Section 40 (notify the NDPC within 72 hours; notify
affected data subjects where the breach is likely to result in high risk).

- Preset: `NDPRBreachReport` from `@tantainnovative/ndpr-toolkit/presets`
- Run locally: `npm install && npm run dev`

Open in StackBlitz:
https://stackblitz.com/github/mr-tanta/ndpr-toolkit/tree/main/examples/stackblitz/breach
