# NDPR ROPA — StackBlitz demo

Minimal Next.js 15 + React 19 scaffold that boots the `NDPRROPA` preset.
Build the Record of Processing Activities every controller and processor
must maintain; state is held in-memory for this demo.

Covers NDPA 2023 Section 29 (record-keeping obligations).

- Preset: `NDPRROPA` from `@tantainnovative/ndpr-toolkit/presets`
- Run locally: `npm install && npm run dev`

Open in StackBlitz:
https://stackblitz.com/github/mr-tanta/ndpr-toolkit/tree/main/examples/stackblitz/ropa
