# NDPR Data Subject Rights — StackBlitz demo

Minimal Next.js 15 + React 19 scaffold that boots the `NDPRSubjectRights`
preset. The page renders the public-facing rights portal so a data subject
can submit access, rectification, erasure, portability, restriction,
objection, or consent-withdrawal requests. Submitted payloads are logged to
the browser console.

Covers NDPA 2023 Sections 34&ndash;38.

- Preset: `NDPRSubjectRights` from `@tantainnovative/ndpr-toolkit/presets/dsr`
- Run locally: `npm install && npm run dev`

Open in StackBlitz:
https://stackblitz.com/github/mr-tanta/ndpr-toolkit/tree/main/examples/stackblitz/dsr
