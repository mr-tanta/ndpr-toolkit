# NDPR Cross-Border Transfer — StackBlitz demo

Minimal Next.js 15 + React 19 scaffold that boots the `NDPRCrossBorder`
preset. Record international data transfers and the safeguard relied on for
each one; state is held in-memory for this demo.

Covers NDPA 2023 Sections 41&ndash;43 (cross-border transfer rules and
approved safeguards).

- Preset: `NDPRCrossBorder` from `@tantainnovative/ndpr-toolkit/presets`
- Run locally: `npm install && npm run dev`

Open in StackBlitz:
https://stackblitz.com/github/mr-tanta/ndpr-toolkit/tree/main/examples/stackblitz/cross-border
