# NDPR Lawful Basis — StackBlitz demo

Minimal Next.js 15 + React 19 scaffold that boots the `NDPRLawfulBasis`
preset. Add processing activities and map each one to a lawful basis;
state is held in-memory for this demo.

Covers NDPA 2023 Section 25(1)(a)&ndash;(f) (consent, contract, legal
obligation, vital interests, public task, legitimate interests).

- Preset: `NDPRLawfulBasis` from `@tantainnovative/ndpr-toolkit/presets`
- Run locally: `npm install && npm run dev`

Open in StackBlitz:
https://stackblitz.com/github/mr-tanta/ndpr-toolkit/tree/main/examples/stackblitz/lawful-basis
